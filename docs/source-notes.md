# Source Notes

This repo should prefer official sources because design internship pages go stale quickly.

## Source Priority

1. Company or program application page.
2. Company university recruiting page.
3. Organization-run fellowship or apprenticeship page.
4. Design-specific job board.
5. General job board search page.

## Verification Routine

For each update:

1. Open the source URL.
2. Confirm the role, program, or board still exists.
3. Confirm status and deadline.
4. Update `last_checked`.
5. Move expired opportunities to `ARCHIVE.md` when useful for future cycles.
